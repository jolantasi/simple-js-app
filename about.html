About This Project

Welcome to **Pokédex**, a simple web application that displays Pokémon details dynamically using JavaScript and Bootstrap.

Features:

  • Fetches Pokémon data from an API
  • Displays Pokémon images, names, and details
  • Responsive Bootstrap design
  • Interactive modal pop-ups for Pokémon details

Tech Stack:

  • HTML, CSS, Bootstrap for styling
  • JavaScript for fetching and handling data
  • PokéAPI for real-time Pokémon data

Credits:

This project uses the free PokéAPI to fetch Pokémon data.
